# Schreiben auf Deutsch - AI-Powered German Writing Practice

**Schreiben auf Deutsch** is a modern, open-source Android application designed to bridge the gap between passive German learning and active writing proficiency. Built with a focus on real-world scenarios, the app serves as a personal AI tutor for German learners of all levels (A1-C1).

## 🚀 Key Features

- **Custom Task Generation:** Create writing exercises based on specific topics, CEFR levels, and examination tracks (Goethe, Telc, DTB).
- **Real-Time Validation:** Get instant visual feedback as you type to catch errors early.
- **Pedagogical AI Feedback:** Receive detailed explanations of grammar mistakes, vocabulary improvements, and phrasing tips in your native language.
- **Dual-Provider Strategy:** Leverages Google Gemini 1.5 Flash and Groq (Llama 3.1) for high-quality, reliable AI generation.
- **Privacy-First:** No accounts required. All your writing history and progress are stored locally on your device.

## 🛠 Technical Architecture

- **UI:** 100% Jetpack Compose with Material 3 Dynamic Theming.
- **Pattern:** MVVM (Model-View-ViewModel) with a clean Repository layer.
- **Navigation:** Modern Navigation 3 implementation with adaptive layout support.
- **Persistence:** Room Database for local data and DataStore for user preferences.
- **Modern Android:** Official Splash Screen API, Edge-to-Edge support, and ProGuard/R8 optimized.
